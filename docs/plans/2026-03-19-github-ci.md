### Task 2: Add The GitHub Actions CI Workflow

**Files:**
- Create: `.github/workflows/ci.yml`

**Step 1: Write the workflow**

Create `.github/workflows/ci.yml` with this exact structure:

```yaml
name: CI

on:
  pull_request:
  push:
    branches:
      - master

concurrency:
  group: ci-${{ github.workflow }}-${{ github.event.pull_request.number || github.ref }}
  cancel-in-progress: true

jobs:
  format:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: "3.10"
      - run: python -m pip install --upgrade pip
      - run: python -m pip install -r requirements-dev.txt
      - run: python -m ruff format --check .

  lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: "3.10"
      - run: python -m pip install --upgrade pip
      - run: python -m pip install -r requirements-dev.txt
      - run: python -m ruff check .

  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: "3.10"
      - run: python -m pip install --upgrade pip
      - run: python -m pip install -r requirements-dev.txt
      - run: python -m pytest -q
```

**Step 2: Verify the CI command set in an isolated Python 3.10 venv**

Run:

```bash
tmpdir="$(mktemp -d)"
python3.10 -m venv "$tmpdir"
"$tmpdir/bin/python" -m pip install --upgrade pip
"$tmpdir/bin/python" -m pip install -r requirements-dev.txt
"$tmpdir/bin/python" -m ruff format --check .
"$tmpdir/bin/python" -m ruff check .
"$tmpdir/bin/python" -m pytest -q
rm -rf "$tmpdir"
```

Expected: all commands PASS locally inside the temporary environment, proving the CI commands are valid without polluting the developer's default Python environment.

**Step 3: Commit the workflow**

Run:

```bash
git add .github/workflows/ci.yml
git commit -m "ci: add GitHub Actions CI workflow"
```

Expected: one commit adding only the workflow file.
